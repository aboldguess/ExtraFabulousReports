Instructions

{# Provide step-by-step guidance for new users #}

This application lets teams create and manage LaTeX reports collaboratively. Follow these steps to get started:

  • Use the Register link to create an account then log in.
  • Navigate to the Documents page to create or edit reports.
  • Upload images using the editor's upload form and reference them in your LaTeX.
  • Include figures with captions using {{'{{figure:path|caption|label}}'}} and reference them later via {{'{{ref:label}}'}}.
  • Administrators can fine-tune the LaTeX look and feel on the House Style page.
